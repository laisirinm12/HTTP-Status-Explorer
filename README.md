# HTTP Status Explorer

This is a simple **HTTP Status Explorer** built using **HTML, CSS, JavaScript and Express.js**.  
It allows users to explore different HTTP status codes by clicking on them and viewing their description and example.

The interface groups status codes into categories so that:
- Users can quickly identify **Success, Redirect, Client Error and Server Error codes**.
- Clicking a code displays **its meaning, description and a real-world example**.

## Features

- Interactive status code explorer
- Color-coded HTTP status categories
- Clickable status code buttons
- Descriptions and examples for each code
- Glassmorphism UI with animated gradient background
- Developer-style typography using **Fira Code**

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js

## Purpose

This project was created as a small full-stack exercise to practice:
- Express routing and API structure
- Organizing backend code using routes and data modules
- Fetch API communication between frontend and backend
- Building an interactive developer-focused UI
